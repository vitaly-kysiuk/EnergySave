# EcoControl [beta]

Системна утиліта для моніторингу стану заряду акумулятора та керування енергоспоживанням Windows.

### Основні можливості:
*   Динамічне керування лімітами потужності процесора через `PowerWriteDCValueIndex`.
*   Вивід поточної напруги в реальному часі.
*   Збір даних про стан акумулятора за допомогою **WMI**.

### Технології:
*   **C# / WinForms**
*   **WinAPI (P/Invoke)**.
*   **WMI (System.Management)**.
