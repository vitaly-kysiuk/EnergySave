# OnlyEnergySave

Системна утиліта для моніторингу стану заряду акумулятора та керування енергоспоживанням Windows.

### 🚀 Основні можливості:
*   **CPU Power Management:** Динамічне керування лімітами потужності процесора через `PowerWriteDCValueIndex`.
*   **Voltage Monitoring:** Вивід поточної напруги в реальному часі.
*   **Battery Analytics:** Збір даних про стан акумулятора за допомогою **WMI**.

### 🛠 Технології:
*   **C# / WinForms**
*   **WinAPI (P/Invoke)** — робота з некерованою пам'яттю та системними GUID.
*   **WMI (System.Management)** — отримання апаратних метрик.
